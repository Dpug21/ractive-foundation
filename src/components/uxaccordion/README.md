##Accordion demo

#### Basic usage
```
<uxaccordion>
	<uxaccordionitem>
		<uxanchor>Testing an anchor1</uxanchor>
		<uxcontent>Accordion 2: Lorem ipsum</uxcontent>
	</uxaccordionitem>

	<uxaccordionitem active="true">
		<uxanchor>Testing an anchor2</uxanchor>
		<uxcontent>Accordion 2: Dolor sit amet</uxcontent>
	</uxaccordionitem>
</uxaccordion>
```
